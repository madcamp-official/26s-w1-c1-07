# MADPUMP 디자인 아이디어 풀

자체 발상(self) + 웹 리서치(web)로 수집한 20개 후보를 유사 컨셉 병합 후 14개로 정리하고, 다양성 축 점검에서 얇았던 '미래' 축을 보충 아이디어 2개(P15, P16)로 채워 총 16개를 확정했다. 각 디자인 루프에서 여기서 하나를 선정하거나 새 아이디어를 추가한 뒤 아래 '사용 기록'을 갱신한다.

병합 내역:
- P02 ← 심야 격투 브라운관(self) + FAMICOM VS. MODE(web)
- P03 ← 스포츠 생중계(self) + Broadcast Clarity(web)
- P06 ← GYM CLASS(self) + Zero-Friction Doodle Lobby(web)
- P10 ← Neo-Brutal Duel HUD(web) + Neo-Brutalist Arena(web) + Neo-Brutal Arcade(web)
- P12 ← Synthwave Arcade Cabinet(web) + NEON COIN-OP(web)

### P01. 펌프장 계기판 [출처: self] [상태: 미사용]
- 축: 다크(중성) / 밀도 중 / 산업 레트로 / 진지함
- 컨셉: 산업용 압력계·유압 펌프장의 아날로그 계기판을 그대로 게임 UI로 가져온다. '펌핑' 대결이라는 게임 정체성과 물리적으로 직결되며, q/w·u/i 두 버튼 조작이 실제 밸브 레버 2개를 잡는 감각으로 번역된다. 바늘이 레드존을 넘어 떨리는 순간의 긴장감이 승부의 클라이맥스를 시각화한다.
- 비주얼 토큰: 팔레트 #1A1D21(기계 하우징 차콜), #E8E4D8(계기판 아이보리), #C9302C(레드존), #E8A33D(경고 앰버), #3F6C51(안전 그린). 타이포: DIN/Bebas 계열 컨덴스드 숫자 + 스텐실 라벨(REV., PSI, MAX), 각인/음각 처리. 형태: 원형 게이지, 리벳 금속 패널, 베벨 물리 버튼, 노랑-검정 해저드 스트라이프. 모션: 바늘 스프링 오버슈트, 임계치 초과 시 패널 진동 + 적색 경광등 점멸, 릴레이 딸깍 즉발 스냅 트랜지션.
- 게임 장면: 숫자 맞추기 펌핑 대결 — 좌우 대칭의 거대 원형 압력 게이지 2개, 연타가 바늘을 밀어올린다. 목표 숫자는 레드존 경계에 스텐실 각인. 목표 근접 시 게이지 유리에 김이 서리고 파이프에서 증기 분출. 승리 시 상대 게이지 유리가 '펑' 금 가는 연출.

### P02. 심야 격투 브라운관 [출처: self+web] [상태: 미사용] (병합: 심야 격투 브라운관 + FAMICOM VS. MODE)
- 축: 다크 / 밀도 중 / 레트로 / 진지+장난 혼합
- 컨셉: 90년대 오락실 CRT 모니터 + 아케이드 격투 게임의 화면 언어. 1:1 대결·2버튼 초간단 조작이 스트리트파이터식 'VS 화면·체력바·라운드 콜' 문법과 정확히 겹치고 레딧 레트로 게이머 감성에 즉시 통한다. 이펙트 강도는 스펙트럼으로 조절 가능: 한쪽 끝은 NES/패미컴 순정 모드(화면당 4~5색 제한, 8x8 비트맵 폰트 단일 사용, blink와 팔레트 스왑만 허용, 글로우/글리치 전면 금지 — 구현 난도 최저), 반대쪽 끝은 CRT 풀옵션(스캔라인, 배럴 왜곡, 인광 번짐, 수직동기 글리치). P1=블루/P2=레드 고정 배색, 이니셜 3글자 하이스코어 테이블 등 고전 관례를 인용한다.
- 비주얼 토큰: 팔레트 #0B0B14(CRT 블랙), #FF2E63(네온 마젠타), #08F7FE(전자빔 시안), #FFE45E(코인 옐로), #7CFC00(인광 그린); 순정 모드는 #000000/#FCFCFC/#E40058/#3CBCFC/#F8B800의 NES 5색. 타이포: Press Start 계열 픽셀 비트맵 대문자 + KO 텍스트는 크롬 아케이드 로고체. 형태: 하드 픽셀 엣지, 8px 그리드 스냅, 둥근 모서리 금지, CRT 배럴 왜곡 + 비네팅, 픽셀 더블라인 프레임, '▶' 픽셀 커서. 모션: 스캔라인 상시 오버레이, 히트 시 1프레임 화이트 플래시 + 화면 흔들림, 수직 롤링 글리치 전환, 12fps 스텝 프레임, 'PUSH START' 500ms blink.
- 게임 장면: 펜싱 가위바위보 — CRT 곡면 안 픽셀아트 결투장, 상단에 격투게임식 체력바 2개가 마주 본다. 'ROUND 1 — FIGHT!' 쾅 등장, 카운트다운은 팔레트 스왑(흰→노랑→빨강). 찌르기 성공 시 흑백 반전 1프레임 + 'HIT' 픽셀 폭발, 승리 시 KO 글자가 배럴 왜곡을 뚫고 튀어나오고 스캔라인이 지지직 흐트러진다. 결과 후 'TODAY'S BEST' 이니셜 3글자 랭킹 테이블이 스크롤 업.
- 참고: https://eirifu.wordpress.com/2025/02/21/a-style-guide-for-nes-inspired-pixel-art-in-your-retro-game/ , https://www.gameuidatabase.com/index.php?scrn=53 , https://www.gameuidatabase.com/index.php?scrn=136

### P03. 스포츠 생중계 — Broadcast Arena [출처: self+web] [상태: 미사용] (병합: 스포츠 생중계 + Broadcast Clarity)
- 축: 다크 네이비 / 밀도 중 / 현대 / 진지함
- 컨셉: 월드컵·UFC 방송 그래픽 패키지와 발로란트 중계 HUD의 진화(불투명 육중한 패널 → 컴팩트·반투명 계층 구조)를 결합해, 모든 대결 화면을 'e스포츠 공식 중계'로 프레이밍한다. 분반 대항전을 '우리 반 vs 너네 반' 국가대항전으로 격상시키고, 리더보드·매칭·결과 화면이 순위표·대진표·MVP 스탯 그래픽으로 자연스럽게 번역된다. 좌우 완전 대칭 미러 구조(파랑 vs 빨강), 게이지를 한곳에 모아 시선 이동 최소화(SF6 아이트래킹 검증 원칙). 관전자도 즐거운 화면이라 캠프 공용 스크린 송출·클립/스트리밍 확산에 최적.
- 비주얼 토큰: 팔레트 #0E1E3C(중계 네이비)/차콜 베이스, #F5F7FA(방송 화이트), #E4B33C(트로피 골드), 팀 컬러 듀얼 #00D1FF(블루) vs #FF4655(레드). 타이포: 초고밀도 익스텐디드/컨덴스드 볼드 대문자 + 이탤릭 기울임 속도감, 숫자는 타뷸러 모노. 형태: 스큐 -12° 평행사변형 로워서드, 반투명 글래스 패널(blur + 8% white), 1px 디바이더, 상단 중앙 스코어 pill, 하단 스크롤 티커. 모션: 로워서드 와이프 인, 플립보드 점수 카운터, 리플레이 화이트 플래시 와이프, 승부 순간 슬로모션 + 골드 파티클, 승부 순간에만 발광하는 글로우 스트로크.
- 게임 장면: 총알 피하기 — 상단 중앙 반투명 스코어보드(닉네임·아바타·라운드·생존 시간), 좌상단 'LIVE' 적색 점 점멸. 두 플레이어 화면이 팀 컬러 듀오톤 분할 동시 중계, 좌우 하단엔 q/w·u/i 키 입력이 점등되는 '선수 시점 인디케이터'. 아슬한 회피엔 노란 서클 하이라이트, 피격 순간 리플레이 와이프 → 슬로모션 → MVP 스탯 카드(정확도·반응속도)로 마무리.
- 참고: https://medium.com/@janushshah/the-5-year-journey-of-valorants-broadcast-ui-from-clutter-to-clarity-f0bc4b60c46d , https://playvalorant.com/en-us/news/game-updates/preview-the-future-of-valorant-s-interface/ , https://sunstrikestudios.com/en/blog/HUD_design_in_games/

### P04. 찢어진 노트 결투장 [출처: self] [상태: 미사용]
- 축: 밝음(저채도) / 밀도 중 / 아날로그(무시대) / 장난스러움(온기)
- 컨셉: 수업 시간에 노트 귀퉁이에 그려서 짝꿍과 하던 낙서 게임(종이축구, 빙고)을 디지털로 승격. 캠프 대학생의 '강의실에서 몰래 하는 대결' 정서를 정확히 찌르고, 하이테크 플랫폼들 사이에서 극단적으로 차별화된 아날로그 촉감을 준다. 승부 기록이 연필 자국으로 남는다는 서사가 리더보드에 온기를 부여한다. (P06 스티커 맥시멀리즘과 달리 저채도·연필 질감·조용한 톤이 핵심.)
- 비주얼 토큰: 팔레트 #F6F1E3(누런 갱지), #2B2B2B(연필 그라파이트), #3E63C4(볼펜 블루), #D94F30(빨간 채점펜), #F2C230(형광펜 옐로). 타이포: 연필 질감 손글씨 헤드라인 + 노트 줄에 맞춘 저대비 세리프 본문, 점수는 바를정(正)자 태리마크. 형태: 손으로 그린 흔들리는 테두리, 마스킹테이프 버튼, 찢어진 종이 결 카드, 노트 스프링 구멍과 파란 괘선 배경. 모션: 페이지 넘김 전환, 연필 스트로크 드로잉 등장, 지우개 벅벅 피격 이펙트, 승리 시 빨간 채점 동그라미 + 별 낙서.
- 게임 장면: 숫자 맞추기 펌핑 대결 — 갱지 위 연필 온도계 낙서 2개, 연타마다 형광펜 칠이 차오른다. 목표 숫자는 빨간 채점펜 동그라미로 상단에. 오버슈트하면 연필 선이 삐져나가며 '아깝다!' 손글씨, 승리하면 상대 온도계에 빨간 X, 내 쪽엔 100점 동그라미와 별표가 슥슥 그려진다.

### P05. PRESSURE — Swiss Brutalist Scoreboard [출처: self] [상태: 미사용]
- 축: 밝음(오프화이트) / 미니멀 / 현대 / 진지함
- 컨셉: 타이포그래피가 곧 UI인 초미니멀 방향. 스위스 그리드 + 브루탈리즘으로 화면의 90%를 거대한 숫자·닉네임·랭크 타이포가 차지한다. '점수와 이름만 남는다'는 랭킹 문화의 본질을 드러내고, 긴장감은 색이 아니라 크기 변화로 표현 — 카운트다운이 다가올수록 숫자가 화면을 압도하고 자간이 조여든다. q/w·u/i도 화면 좌우 하단의 거대한 활자로만 존재. 숫자·기호 중심 언어라 해외 확산 시 언어 장벽이 거의 없다.
- 비주얼 토큰: 배경 #F4F1EA(웜 오프화이트) / 잉크 #111111 / 강조색은 시그널 레드 #FF2E00 단 하나. 타이포: Helvetica Now Display/Neue Haas Grotesk, 숫자는 tabular lining 120–400px. 형태: 12컬럼 하드 그리드, 1px 헤어라인 룰러, 라운딩 0px, 그림자 없음. 모션: 활자 스케일/트래킹 트윈(cubic-bezier(0.9,0,0.1,1))만 사용, 호버 = 흑백 반전 블록.
- 게임 장면: 숫자 맞추기 펌핑 대결 — 1px 검은 선으로 세로 이분할, 좌우에 300px 숫자가 실시간 카운트업. 목표 숫자는 상단 중앙에 24px로 작게 고정(정보 위계의 역전). 목표 근접 시 자간이 좁아지며 미세하게 떨리고, 오버슈트 순간 시그널 레드 반전 + 'BUST'. 승자 쪽은 흑백 반전, 패자 숫자는 회색으로 바래며 무너져 내린다.

### P06. GYM CLASS — 스티커 두들 플레이그라운드 [출처: self+web] [상태: 미사용] (병합: GYM CLASS 스티커 맥시멀리즘 + Zero-Friction Doodle Lobby)
- 축: 밝음 / 맥시멀 / 현대(학교 정서) / 장난스러움
- 컨셉: 노트 낙서·스티커·마스킹테이프·손그림 화살표가 겹겹이 붙은 콜라주 맥시멀리즘에, skribbl.io/JKLM식 '60초 안에 시작' 제로 프릭션 UX를 결합. 진지한 승부를 우스꽝스러운 포장지에 싸는 반전 — 겉은 장난 같지만 리더보드 숫자는 냉정하다. 랜딩=로비: 닉네임 → 좌우 화살표 즉석 아바타(펌핑 근육 캐릭터) → 거대 Play 버튼이 한 화면에서 끝나고, 코드방은 URL 자동 복사 토스트. 분반별 컬러 스티커, 승리 시 상대 프로필에 붙는 '참 잘했어요' 도장, 연승 배지가 지저분하게 쌓이는 프로필 등 커뮤니티 문화 장치가 핵심. q/w·u/i 안내는 텍스트 튜토리얼 대신 로비에서 즉시 눌러볼 수 있는 인터랙티브 키캡.
- 비주얼 토큰: 배경 크라프트지 #E8D9B5 또는 모눈종이/밝은 단색(하늘색·크림). 팔레트: 형광 하이라이터 4색(#FFF200, #FF6EC7, #7FFF6A, #5CE1E6) + 볼펜 블루 #1B3FAB. 타이포: 손글씨 디스플레이(Gaegu/Caveat 계열) + 둥근 산세리프 본문. 형태: 흰 테두리 스티커 다이컷, -3°~3° 비뚤어진 회전, 마스킹테이프 카드, 오프셋 하드섀도, Q/W/U/I 실물 키캡 토큰(눌리면 2px 가라앉음). 모션: 스티커 철썩 squash&stretch, 도장 쾅 임팩트, hover 시 wiggle, '코드 복사됨!' 토스트.
