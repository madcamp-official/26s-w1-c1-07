## §1. 디자인 시스템 — "NEON COIN-OP"

컨셉 한 줄: **80년대 아케이드 캐비닛 × 신스웨이브.** 딥퍼플블랙 바탕 + 네온 1~2색 악센트 +
소실점 그리드. 전 플로우를 아케이드 서사로 묶는다 — 매칭=INSERT COIN, 대기=attract mode,
리더보드=HI-SCORE 테이블, 결과=하이스코어 등록 화면. 루프 1(밝은 크림+하드엣지)과 정반대 극의
다크+글로우 룩.

- **P1 vs P2 색 구분(SPEC 필수, 주석 16:1713)**: 모든 인게임·HUD·프로필·패드에서
  좌=`--p1` 시안 / 우=`--p2` 핑크 절대 고정. 채움이 필요한 곳은 `--p1-dim`/`--p2-dim` 바탕 +
  플레이어색 2px 보더 + 플레이어색 텍스트(다크 위 네온 원칙 — 큰 면적을 순색으로 채우지 않는다).
- **네온 절제 원칙**: 화면당 발광 요소는 3군데 이하. 나머지는 다크 표면 + muted 텍스트로 눌러
  네온이 항상 주인공이 되게 한다. `--accent`(옐로)는 "지금 눌러야 할 것" 하나에만.
- 글로우 공식(전역 상수): 텍스트 `text-shadow: 0 0 4px currentColor, 0 0 18px currentColor;`
  박스 `box-shadow: 0 0 6px <색>, 0 0 20px rgba(<색>,0.45), inset 0 0 8px rgba(<색>,0.25);`

### 1.3 형태 언어

- **radius**: 기본 `0px`(각진 사각 + 네온 보더가 기본형). 예외 — 아케이드 버튼(코인 버튼)만
  완전 원형, 키캡은 `6px`. 부드러운 라운딩 카드 금지.
- **border**: 표면 구획은 `1px solid rgba(211,0,197,0.35)`(퍼플 헤어라인), 강조 구획은
  `2px solid` 플레이어색/포인트색 + 글로우. **코너 브래킷**: 히어로 패널(모달·스테이지)은
  네 모서리에 12px `L`자 브래킷(2px, 포인트색)을 덧대 CRT 오버레이 프레임 느낌.
- **shadow**: 하드 오프셋 섀도 금지 — 이 시안의 그림자는 전부 **글로우**(§1.1 공식).
  깊이는 배경 레이어 단차(`--bg` → `--bg-raised` → `--surface`)로만 표현.
- **스캔라인 오버레이**: 전역 고정 오버레이 1장 —
  `repeating-linear-gradient(0deg, rgba(0,0,0,0.22) 0 1px, transparent 1px 3px)` + 노이즈 그레인
  2% + 가장자리 비네팅. `pointer-events:none`, 모든 화면 공통.
- **소실점 그리드**: 메인 계열 화면(S1·S2·S5·S8) 하단 1/3에 원근 그리드
  (`--accent2` 1px 선, 수평선 쪽으로 수렴, 지평선엔 핑크→퍼플 글로우 밴드). 인게임에서는
  스테이지 밖 배경에만 옅게(20% 투명) 유지.
- **CRT 베젤**: 인게임 `game-stage`는 `--bg-raised` 필드를 `#000` 두꺼운 베젤(12~16px)이 감싸고
  베젤 모서리만 radius 12px(브라운관 실루엣) + 안쪽 1px 시안 헤어라인.
- **램프/코인 모티프**: 라운드 승수는 아케이드 캐비닛의 원형 램프(점등=플레이어색 글로우,
  소등=`--surface-deep` + 헤어라인)로. 크레딧/코인 아이콘은 옐로 원 + `¢`(코인 슬롯 인용).

### 1.4 모션 규칙

- **네온 점등(sign-on)**: 화면/패널 등장 시 헤드라인이 `off → 2회 플리커 → on`(총 400ms,
  steps 기반 opacity 0/1 교차). 페이드 인 금지 — 형광등이 켜지는 리듬.
- **블링크**: "INSERT COIN"·대기 문구는 1s 주기 `steps(1)` 점멸(50% duty). 부드러운 pulse 금지.
- **호버 플리커**: 버튼 hover 시 보더 글로우가 80ms 1회 플리커 후 밝기 1.4배 유지. press 시
  글로우 소멸 + 표면이 안쪽으로 눌린 톤(`--surface-deep`) 120ms.
- **글리치는 승패 순간에만**: 화면 전환·판정에 크로마틱 어버레이션(시안/핑크 2px 오프셋 잔상)
  + 수평 슬라이스 흔들림 1~2프레임. 평시 글리치 금지(멀미 방지).
- **화면 전환**: CRT 채널 전환 — 검은 수평 롤링 밴드가 위→아래 훑기(220ms) 후 새 화면 sign-on.
- **키 입력 피드백**: 온스크린 키캡 램프가 즉발 점등(0ms) → 80ms 후 소등. 트랜지션 없는 순간 반응.
- **카운트다운 임박(≤5초)**: 타이머 숫자가 옐로→핑크 교차 점멸 + 스테이지 베젤 안쪽 헤어라인이
  심장박동처럼 명멸(600ms 주기).
- `prefers-reduced-motion`: 플리커·점멸·글리치 전부 생략, 정적 글로우와 색/텍스트로만 상태 표현.

### 1.5 컴포넌트 스타일 지침

- **버튼**: 기본형 = 다크 표면 + 2px 네온 보더 + 글로우 + Gugi 한글 라벨(또는 Press Start 2P 영문).
  위계 — Primary(온라인 게임하기, 빠른 시작): `--accent` 옐로 보더·텍스트 + 좌측에 코인 아이콘 /
  Secondary(오프라인 등): `--p1` 시안 보더 / Tertiary(로그아웃 등): 보더 없이 muted 텍스트 +
  hover 시 시안 점등. 파괴적(취소하기): `--error` 보더. **아케이드 코인 버튼**(설정 등 원형 진입점):
  빨간 돔 대신 네온 링 — 원형, 2px 보더 + 중앙 아이콘, press 시 링 글로우 소멸.
- **카드**: `--surface` + 퍼플 헤어라인 보더 + 상단에 **마퀴 스트립**(캐비닛 상단 간판 인용 —
  어두운 띠에 네온 텍스트 제목 + 좌우 끝 소형 램프 2개). 게임 선택 카드는 §2-S8의 캐비닛형.
- **모달**: 오버레이 `rgba(13,2,33,0.82)` + 오버레이에도 스캔라인. 모달 본체는 `--surface` +
  2px 포인트색 보더 + 코너 브래킷 + 상단 마퀴 스트립 제목. 등장은 sign-on 플리커(§1.4).
- **입력**: 터미널 스타일 — `--surface-deep` 바탕 + 하단 2px 시안 라인(보더는 하단만),
  텍스트·caret 시안, placeholder `--text-muted`. focus 시 하단 라인 글로우 + 좌측에 `>` 프롬프트
  문자 점등. 에러 시 라인·텍스트 `--error` + 아래 에러 문구(본문 폰트) + 1회 수평 셰이크.
- **테이블(리더보드)**: HI-SCORE 테이블 문법 — 외곽 2px 시안 보더 + 글로우, 헤더 행은
  Press Start 2P 소형 대문자(RANK/NAME/PLAY/WIN/RATE), 행 디바이더 없이 짝수행
  `--surface-deep`. 순위 1~3 행은 랭크 문자가 `1ST`(옐로)/`2ND`(시안)/`3RD`(핑크) 네온 발광.
  내 행은 좌측 4px 시안 바 + "YOU" 점멸 캡션. 수치는 Press Start 2P(고정폭이라 타뷸러 정렬 자동).
- **키캡(공통 부품)**: radius 6px, 2px 플레이어색 보더, 위에 화살표/아이콘 + 아래 Press Start 2P
  키 문자. P1 키캡 시안 / P2 키캡 핑크. 입력 순간 램프 점등(§1.4) — "캐비닛의 버튼 2개"가
  화면 안에 항상 존재.
- **공통 HUD 프레임(인게임 S9~S12)**: 스테이지 CRT 베젤 상단에 겹쳐 얹는 아케이드 스코어 라인 —
  좌 `hud-profile-p1`(시안: 픽셀 아바타 사각 + 닉네임 + 라운드 램프 열) ↔ 중앙
  `hud-countdown`(Press Start 2P 대형 초 + "TIME" 캡션) ↔ 우 `hud-profile-p2`(핑크 미러).
  내 쪽 프로필엔 시안/핑크 "YOU" 점멸 태그. 라운드 램프는 중앙을 향해 배열.

## §2. 페이지별 계획 (S1~S12)

### S1. 메인 — 비로그인 (`scr-main-out`)
- **기능(SPEC)**: MADPUMP 타이틀 / SIGN IN WITH GOOGLE(구글 로고) / 설정 버튼 / 온라인·오프라인
  게임하기 2버튼. 비로그인 온라인 → S3 모달, 오프라인 → S8.
- **레이아웃**: **attract mode 타이틀 스크린.** 하단 1/3 소실점 그리드 + 지평선 글로우, 그 위
  중앙에 초대형 "MADPUMP" — Gugi 아닌 Press Start 2P… 대신 영문 로고는 **네온 사인 2색 처리**
  (MAD=핑크 / PUMP=시안, 각 글자 sign-on 플리커, 이후 P 하나가 간헐적으로 깜빡이는 고장 램프 개그).
  로고 아래 "1v1 PUMPING DUEL" 퍼플 캡션. 중앙 하단 세로 스택: [온라인 게임하기](`btn-online`,
  옐로 primary — 위에 "INSERT COIN ▶" 점멸 캡션) / [오프라인 게임하기](`btn-offline`, 시안 secondary).
  우상단: [SIGN IN WITH GOOGLE](`btn-google-login`, 다크 표면 + G 로고 + 화이트 헤어라인 —
  구글 로고만은 원색 유지) + 원형 코인 버튼 [설정](`btn-settings`, 톱니). 하단 티커 스트립:
  "Q·W vs U·I — TWO BUTTONS. ONE WINNER." Press Start 2P 소형 스크롤(장식, 기능 아님).

### S2. 메인 — 로그인 후 (`scr-main-in`)
- **기능(SPEC)**: 닉네임 인사말 / 로그아웃 / 설정 / 분반 리더보드(TOP3 플레이수·승수·승률 + 내 등수)
  / 온라인·오프라인 버튼. 빈 데이터 시 빈 상태 정직 표기.
- **레이아웃**: S1 골격에서 로고를 좌측 55% 존으로 옮기고 우측 40%에 **HI-SCORE 패널**.
  우상단 헤더: "OOO님 안녕하세요"를 "PLAYER 1: OOO" 겸용 네온 캡션(시안)으로 + 로그아웃(tertiary)
  + 설정 코인 버튼. 리더보드 카드: 마퀴 스트립 "N분반 HI-SCORE" + §1.5 테이블 — TOP3(`lb-top3`)
  + 하단 분리 스트립 내 등수 1줄(`lb-myrank`, 시안 바 + YOU 점멸). 데이터는
  `computeLeaderboard` 사용, 비면 "NO RECORD — 아직 기록이 없습니다" 점멸 없는 muted 문구.
  게임 버튼 2개는 S1과 동일 위치·위계.

### S3. 로그인 요구 모달 (`modal-login-required`)
- **기능(SPEC)**: "온라인 게임은 로그인이 필요합니다!" 문구 / 모달 내 구글 로그인(성공 시 S6 연속

## §3. 게임 장면 아트 디렉션

### 3.1 게임1 — 숫자 펌핑: "세 개의 스코어보드, 하나의 잭팟"
- **숫자 = 아케이드 스코어 카운터.** 값 변경은 트윈 없이 하드 스텝 교체 + 바뀐 숫자에
  80ms 글로우 버스트(밝기 1.6배 → 원복) — 연타할수록 패널이 번쩍이는 펌핑 감각.
  증가는 패널 위 `▲` 램프, 감소는 `▼` 램프가 1프레임 점등.
- **근접 피드백**: |현재−타겟| ≤ 5면 내 패널 보더가 옐로로 전환, ≤ 2면 보더 글로우가 빠르게
  맥동(300ms) — "잭팟 직전" 긴장. (수치 비표시 — 순수 연출, 로직 비침범.)
- **일치 유지 3초 = 코인 락인**: 일치 순간 패널 위에 소등된 램프 3개가 나타나 1초마다 하나씩
  플레이어색으로 점등(1→2→3). 이탈 시 램프 전체가 글리치와 함께 일괄 소등 — 리셋 즉독.
  3점등 → "LOCKED!" 캡션 + 패널 전체 글로우 → 라운드 결과 오버레이의 WINNER 사인으로 연결.
- 배경: 필드에 초대형 아웃라인 "PUMP" 워터마크(퍼플 4%), 타겟 박스 뒤 방사형 옅은 광.

### 3.2 게임2 — 총알: "네온 탄막, 트레일의 비"
- **총알 = 네온 트레이서**: 옐로 코어 도트(4px) + 뒤로 길게 끄는 그라디언트 트레일
  (옐로→투명, 속도가 빠를수록 트레일이 길다) — 속도 랜덤(행19)이 트레일 길이로 즉독.
  P1/P2 색과 겹치지 않는 옐로로 탄을 통일해 "위험물"이 항상 구분된다.
- **발사**: 포탑이 1프레임 반동 + 총구에 시안 머즐 스파크(십자 광점). 쿨다운은 포탑 옆
  탄약 램프가 소등→재점등(0.5초)되는 것으로 표시 — 장전 완료 순간 짧은 플리커.
- **회피/피격**: 러너는 이동 방향으로 잔상 2개(투명도 계단)를 끈다. 근접 회피 시 그 좌표에
  "CLOSE!" 시안 미니 캡션 0.4초. **피격 순간**: 전체 필드 크로마틱 어버레이션 1프레임 →
  러너 자리에 픽셀 파편 6개가 방사형으로 튀며 소멸 + "HIT!" 핑크 캡션 → WINNER 사인.
  P2 생존 승리 시 하단 레일 전체가 핑크 글로우로 러쉬 점등 + "SURVIVED!" 캡션.
- 타이머 임박 5초: 낙하 공간의 세로 그리드가 핑크 톤으로 상승 + 베젤 헤어라인 명멸(§1.4).

### 3.3 게임3 — 펜싱: "그리드 피스트와 와이어프레임 바다"
- **무대**: 소실점 그리드 질감의 발광 플랫폼(상판 퍼플 그리드 + 옆면 `#000`) — 7칸 경계마다
  눈금 램프. 양끝 낭떠러지 밖은 시안 와이어프레임 파도 2~3겹이 `steps(2)`로 어긋나게 출렁이고,
  플랫폼 끝단 칸은 `--error` 헤어라인 + 점멸 — "여기서 떨어지면 끝" 즉독.
- **검객**: 네온 아웃라인 스틱 파이터(2px 스트로크, 좌=시안/우=핑크, 채움 없음 — 관(tube)으로
  만든 네온 사인 인간). 공격=런지(전방 기울임 + 검 수평 돌출 + 검끝 광점), 회피=상체 후퇴 +
  방패 원호가 잔광과 함께 전개, 무행동=중립 겨눔 자세(느린 idle 미세 흔들림).
- **틱 판정 리듬**: 매 1초 틱마다 ①머리 위 픽셀 칩에 양측 선택 동시 공개 → ②밀리는 쪽이
  1칸 하드 스텝 후퇴(120ms) + 발밑 스파크 2개 + "TOUCHÉ!" 캡션이 승자색으로 점등.
  비김이면 중앙에서 두 검이 교차하며 흰 스파크 십자 + "CLASH" 미니 캡션 + 2px 셰이크.
- **링아웃**: 패자가 2스텝 프레임으로 회전 낙하 → 와이어프레임 파도가 크게 한 번 튀며
  시안 스플래시 라인 방사 → 전체 글리치 1프레임 → "RING OUT!" 초대형 캡션 + WINNER 사인 점등.
- 남은 칸 수는 각 프로필 아래 램프 3개로 병기 — 밀릴 때마다 하나가 소등(글리치 1회).
