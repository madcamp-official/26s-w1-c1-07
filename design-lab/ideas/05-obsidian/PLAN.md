### 1.3 형태 언어

- **45° 코너컷 패널**: 카드/모달/주요 버튼은 `clip-path: polygon(...)`으로 좌상+우하(또는 우상 1곳) 모서리를 12~16px 사선 컷. radius는 사용하지 않는다(`border-radius: 0` 기본, 소형 칩만 2px 허용).
- **1px 발광 보더**: 기본 상태 `1px solid var(--line)`. 활성/호버/포커스 시 진영색 or 시안 1px 보더 + `box-shadow: 0 0 12px -2px <색 40%>` 외광 + `inset 0 0 0 1px <색 25%>`. 두꺼운 보더 금지 — 발광은 얇을수록 고급.
- **헥사곤 티어 엠블럼**: 리더보드 순위·프로필 아바타는 헥사곤 클립(`clip-path: polygon(50% 0,100% 25%,100% 75%,50% 100%,0 75%,0 25%)`). 1위 골드 / 2위 `#B8C4D6`(실버) / 3위 `#B0784A`(브론즈), 그 외 `--line`.
- **코너 브래킷 장식**: 인게임 스테이지와 대형 패널 네 귀퉁이에 `┐└` 형 12px 브래킷(1px, 진영색/시안)을 장식으로. 스캔라인·글리치는 전면 금지 — 클린 글로우만.
- 그림자: 부유 요소(모달)는 `0 24px 64px rgba(0,0,0,.6)` 딥 섀도 1종만. 밝은 드롭섀도 금지.

### 1.4 모션 규칙

- 기본 이징 `cubic-bezier(.16,1,.3,1)`(expo-out), 기본 시간 240ms. 장식적 바운스 금지 — 절제가 톤.
- **슬라이드인 + 글로우 차오름**: 패널 등장은 `translateY(12px)→0` + opacity, 등장 완료 후 보더 글로우가 60ms 늦게 점등(box-shadow transition).
- **VS 와이프**: 매칭 성사 순간 화면 좌에서 시안 사선 패널, 우에서 마젠타 사선 패널이 충돌(각 -12° 스큐, 320ms) 후 중앙 "VS" Orbitron 900 스탬프. 이후 인게임 전환.
- 카운트다운: 매초 숫자가 `scale(1.12)→1` + 글로우 펄스. 마지막 5초는 아레나 보더가 심장박동처럼 명멸(`@keyframes` 1s, 마젠타 틴트).
- 버튼 호버: `translateY(-1px)` + 보더 발광. 클릭: 발광 순간 강화 후 감쇠.
- 승패: 슬로모션 컨셉 — result 오버레이 배경이 승자 진영색으로 600ms에 걸쳐 서서히 물든다(radial-gradient 확장).
- `prefers-reduced-motion` 시 트랜지션 시간 0.01ms로 강등.

### 1.5 컴포넌트 스타일 지침

- **버튼**
  - Primary(온라인 게임하기, 빠른 시작, 확인): surface 배경 + 시안 1px 발광 보더 + 코너컷, 내부 텍스트 2줄 스택(영문 OVERLINE + 한글). 호버 시 `--p1-dim` 필.
  - Secondary(오프라인, 취소, 기본값): `--line` 보더, 발광 없음. 호버 시 보더가 `--text-md`로 밝아짐.
  - Ghost/텍스트(로그아웃): 텍스트 + 호버 언더라인(시안 1px).
  - Google 로그인: 다크 surface 위 흰 G 로고 + "SIGN IN WITH GOOGLE"(Orbitron 500) — 다크 테마 구글 버튼 관례.
- **카드**(게임 선택, 리더보드 패널): surface + 코너컷 + 상단 OVERLINE 레이블 행. 게임 카드 호버 시 카드 전용 액센트 발광 + 내부 미니 비주얼 활성.
- **모달**: 배경 `rgba(6,8,12,.78)` + `backdrop-filter: blur(6px)`. 모달 본체는 코너컷 surface, 상단에 1px 시안 발광 톱라인 + 좌상단 OVERLINE(`PROTOCOL // ...`). 등장 모션 1.4의 슬라이드인.
- **입력**: `--bg-1` 배경, 1px `--line` 보더, 포커스 시 시안 발광 보더 + 캐럿 시안. 라벨은 OVERLINE 스타일. 에러 시 보더/헬프텍스트 `--danger`.
- **테이블(리더보드)**: 헤더는 OVERLINE 11px `--text-md`, 로우는 `--line-dim` 1px 디바이더, hover 로우 `--surface-hi`. RANK 열은 헥사곤 엠블럼, 승률 숫자는 Orbitron tabular. TOP 3 로우는 좌측 2px 골드 인셋 라인. 내 로우는 시안 인셋 라인 + "YOU" 칩.
- **키캡 표기**(인게임 공통): `[Q]` 스타일 — 1px 보더 사각 칩 + Orbitron, 해당 키 눌림 시 진영색 필 점등. playerL=Q/W, playerR=U/I 실제 배정 키 표기(SPEC Q2).

## §3. 게임 장면 아트 디렉션

### 3.1 게임1 — 숫자 펌핑: "주파수 동조"

숫자 맞추기를 **송신 주파수를 타겟에 동조시키는 계기 콘솔**로 연출한다.
- 화면 전체가 관제 콘솔: 중앙 타겟 숫자는 백색 코어 글로우, 좌우 진영 숫자는 각자 진영색. 숫자 변경 시 슬롯 롤 대신 **글로우 잔상 틱**(이전 숫자가 0.15초 dim 잔상으로 남고 새 숫자 점등) — 연타의 타격감을 빛으로.
- 근접도 게이지: 각 콘솔 옆 세로 슬림 바가 |현재-타겟| 역비례로 차오르고, 완전 일치 시 바 끝이 백색 스파크.
- 3초 유지 링이 이 게임의 클라이맥스 비주얼: conic 링이 차오르는 동안 해당 진영색 글로우가 스테이지 절반을 서서히 물들이고, 심박 사운드 느낌의 스케일 펄스. 이탈 시 링이 파편 라인으로 깨지며 소멸 — "아깝다"의 시각화.
- 승리: 링 완성 순간 진영색 플래시 1프레임 → 슬로 페이드로 result.

### 3.2 게임2 — 총알: "빛의 그물"

P07 원안 그대로 — 칠흑 아레나에 시안 vs 마젠타 광점, 총알이 광선 궤적을 남기며 화면이 두 색의 빛 그물로 가득 찬다.
- 총알 = 진영색 광탄(코어 백색 + 외광 진영색). 낙하 궤적에 6~8프레임 길이의 페이드 트레일(CSS/canvas 잔상) — 속도 랜덤이라 트레일 길이도 달라져 위협이 직관적으로 읽힌다.
- 회피자 이동에도 짧은 대시 트레일. 총알이 회피자를 스치는 근접 회피 시 `CLOSE CALL` 이탤릭 Orbitron 레이블이 스치는 지점에 짧게 점등(연출 전용 — 점수 개입 없음).
- 발사 순간 공격자 레일이 반동으로 1px 흔들리고 발사구에 머즐 글로우.
- 마지막 5초: 아레나 4변 보더가 심장박동 명멸(공통 셸 규칙과 동조).
- 피격: 슬로모션 감속 → 피격 링 확산 → 승자 진영색이 아레나를 물들임.

### 3.3 게임3 — 펜싱: "심해 위의 결투장"

어둠 위에 떠 있는 발광 피스트, 그 아래는 심해.
- 무대: 옵시디언 플랫폼 상판에 칸 경계 1px 라인(중립 `--line`), 상판 가장자리에 백색 1px 하이라이트. 각 플레이어 배후 칸들은 잔여 여유에 따라 톤 변화(3칸 안전 → 마지막 1칸 위험 명멸).
- 바다: 플랫폼 좌우 아래로 `#06121E`→`#0A0C10` 심해 그라디언트 + 수평 파도 라인 2~3개가 느리게 시프트(4s ease 루프). 물은 검푸른 무광 — 네온 금지, 심연의 공포가 톤.
- 캐릭터: 진영색 1.5px 네온 라인 스틱 파이터 + 관절 포인트 글로우. 포즈 3종(런지/방패/중립)을 틱마다 스냅 전환(트윈 없음 — 가위바위보의 "동시 공개" 긴장감).
- 판정 연출: 공개 순간 양 선택 칩이 중앙에서 충돌 → 승 상성 칩이 발광, 패 측 캐릭터가 1칸 뒤로 미끄러지며 발밑 칸 라인이 진영색 플래시.
- 링아웃: 패자 라인 파이터가 심해로 낙하하며 글로우가 수면 아래로 잦아들고, 낙하 지점에 백색 물결 링 2개 확산 → 승자 컬러 워시.
