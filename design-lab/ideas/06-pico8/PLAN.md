# PICO-8 PLAYGROUND — 팬시 콘솔 픽셀 팝 (루프 6 · P14)

> 컨셉 한 줄: PICO-8 팬시 콘솔의 16색 고정 팔레트와 게임잼(Ludum Dare/itch.io) 감성을 차용한 밝고 장난기 있는 픽셀 방향. 다크 네온(02)과 정반대로, 채도 높은 다색 팔레트 + 카트리지 진열대 메타포로 "인디 픽셀 게임 플랫폼" 정체성을 즉시 전달한다.
>
> - 기능 정본: `design-lab/SPEC.md` (S1~S12, QA 체크리스트 전부 준수)
> - 시각 정본: 이 문서. 와이어프레임은 "무엇이 있어야 하는지"의 참고일 뿐.
> - 게임 로직: `@shared` (`design-lab/shared`) import — 재구현 금지.

### 1.3 형태 언어 (radius/border/shadow)

- **radius 0 + 픽셀 노치 코너**: 모든 카드/모달/버튼은 `border-radius: 0`. 대형 패널은 네 모서리를 4px×4px 계단 클립(`clip-path: polygon(...)`)으로 깎아 "픽셀 라운드" 흉내 — 진짜 라운드 금지.
- **아웃라인**: 모든 인터랙티브 요소와 카드에 `2px solid #000000`(스케일 감안해 1px 대신 2px 채택 — 8px 그리드의 논리 1px). 활성/포커스 시 아웃라인 바깥에 진영색/오렌지 2px 링 추가.
- **하드 드롭섀도**: `box-shadow: 4px 4px 0 #000000`. blur 0 고정. 모달 등 부유 레이어는 `8px 8px 0 rgba(0,0,0,.55)`.
- **8px 그리드 스냅**: 여백·크기 전부 8의 배수. 아이콘/장식 스프라이트는 인라인 SVG `<rect>` 조합(shape-rendering: crispEdges) 또는 box-shadow 픽셀아트로 제작, `image-rendering: pixelated` 전역 적용(canvas 포함).
- **콘솔 스크린 프레임**: 인게임 스테이지(S9~S12)는 정사각~4:3 비율 "스크린"을 화면 중앙에 두고 바깥은 `--bg` 베젤. 스크린 테두리는 `#000` 4px + 안쪽 `--surface-3` 2px 이중 프레임, 좌하단에 `MADPUMP-8` 각인(픽셀폰트 8px, `--text-soft`).
- **카트리지 모티프**: 게임 카드(S8)는 상단에 라벨 스티커 영역 + 하단에 단자 빗살(세로 스트라이프) 장식이 있는 카트리지 실루엣.

### 1.4 모션 규칙

- 트윈 대신 **스텝 애니메이션**: 모든 keyframes는 `steps(n)` 이징 — 저프레임 칩튠 감성. 기본 `steps(3)` 120ms. 부드러운 ease/스프링 전면 금지.
- **버튼 press**: 본체 `translate(2px, 2px)` + 섀도 `2px 2px 0`으로 축소(물리적으로 눌림). hover는 `translate(-2px,-2px)` + 섀도 `6px 6px 0` 확대.
- **blink 커서**: 선택 가능/대기 문구 앞 `▶`(오렌지)가 530ms 간격 step 점멸. 매칭 대기 중 말줄임표 `...`도 글자 단위 점멸.
- **3프레임 픽셀 폭발**: 피격·팝 이펙트는 스프라이트 3프레임(소→대→파편) step 재생 180ms. 화면 흔들림은 1프레임 `translate(4px,0)` 왕복 2회.
- **화면 전환**: 팔레트 스왑 플래시(배경이 `#000` 1프레임) 후 새 화면이 8px 단위로 아래서 스냅 등장(`steps(4)` 200ms). 페이드 금지.
- **카운트다운 마지막 3초**: 숫자가 `#FFF1E8→#FFEC27→#FF004D` 팔레트 스왑 + 매초 scale 1.25→1 step 펄스.
- `prefers-reduced-motion` 시 흔들림/점멸 제거, 전환은 즉시 스냅.

### 1.5 컴포넌트 스타일 지침

- **버튼**: `--surface` 바탕 + 2px 검정 아웃라인 + 4px 하드섀도 + Do Hyeon 18px(한글) 또는 Press Start 2P 12px(영문). 주 CTA(온라인/오프라인 게임하기, 확인)는 `--accent` 바탕 + `#000` 텍스트. 보조(취소하기, 기본값)는 `--surface-3` 바탕 + `--text`. hover 시 좌측에 `▶` 커서 출현. disabled는 `--surface-3` 바탕 + `--text-soft` + 섀도 제거.
- **카드**: `--surface`(또는 문맥색) 바탕 + 픽셀 노치 코너 + 검정 아웃라인 + 4px 섀도. 헤더는 상단 검정 2px 라인 아래 픽셀폰트 오버라인. 게임 카드는 카트리지 실루엣(1.3) + 게임 시그니처 컬러 라벨.
- **모달**: 배경 오버레이 `rgba(0,0,0,.7)` + 2px 체커 도트 패턴. 모달 본체는 카드와 동일하되 8px 딥섀도, 상단에 픽셀폰트 타이틀 바(`--bg` 바탕 띠). 등장은 스냅 팝(scale steps(2): 0.9→1).
- **입력**: `#000` 바탕 + `--text` 텍스트 + 2px `--surface-3` 보더, 포커스 시 보더 `--accent` + 커서 캐럿 점멸 흉내(기본 캐럿 색 `--accent-2`). placeholder `--text-soft`. 에러 시 보더 `--danger` + 아래 `--danger` 12px 문구 + 1프레임 좌우 셰이크.
- **테이블(리더보드)**: 검정 2px 괘선, 헤더 행 `--bg` 바탕 픽셀폰트 10px, 본문 Do Hyeon 14px. 1~3위는 행 좌측에 픽셀 트로피/별 스프라이트(옐로·라이트그레이·오렌지) + 순위 숫자 픽셀폰트. 내 등수 행은 `--accent` 2px 링 + `▶` 커서로 하이라이트. HI-SCORE 테이블 관례로 헤더에 `RANKING` 표기.
- **온스크린 키패드**: 실물 키캡 스타일 — `--surface-3` 바탕 사각 키캡에 배정 키 문자(Q/W/U/I, 픽셀폰트) + 화살표/아이콘 스프라이트. 실제 키 입력과 1:1로 press 모션(2px 하강) 동기화. 진영색 2px 링으로 소유자 표시.

## §2. 페이지별 계획 (S1~S12)

공통 셸: 화면 전체가 `--bg` 콘솔 스크린. 최상단에 얇은 검정 스테이터스 바(좌: `MADPUMP-8 v1.0` 픽셀폰트 8px / 우: 세션 상태) — 팬시 콘솔의 에디터 바 인용. `window.__MADPUMP__` 디버그 브리지는 화면 전환·게임 틱마다 갱신.

### S1. 메인 — 비로그인 (`scr-main-out`)
- 기능(SPEC): MADPUMP 타이틀 / SIGN IN WITH GOOGLE(`btn-google-login`) / 온라인(`btn-online`)·오프라인(`btn-offline`) 게임하기 / 설정(`btn-settings`). 비로그인+온라인 → S3 모달.
- 레이아웃: 중앙 정렬 타이틀 스크린 — 게임 부트 화면 연출. `MADPUMP` 픽셀폰트 초대형(56px)을 글자별 팔레트 다색(오렌지/옐로/블루/그린 교차)으로, 아래 `PRESS ANY BUTTON TO PUMP` 옐로 점멸 태그라인. 타이틀 좌우에 8x8 픽셀 별·하트 스프라이트 부유(step 애니). 그 아래 세로 스택으로 `▶ 온라인 게임하기`(오렌지 CTA) / `▶ 오프라인 게임하기`(퍼플) 메뉴 — 고전 타이틀 메뉴 문법. 우상단에 구글 로그인 버튼(흰 바탕 + G 픽셀 로고 + `SIGN IN WITH GOOGLE`)과 톱니 스프라이트 설정 버튼(원형 대신 사각 키캡).

### S2. 메인 — 로그인 후 (`scr-main-in`)
- 기능: 닉네임 인사말 / 로그아웃 / 분반 리더보드(TOP3 플레이수·승수·승률 + 내 등수, 빈 상태 정직 표시) / 온라인·오프라인 / 설정.
- 레이아웃: S1의 타이틀+메뉴를 좌측 2/3로 옮기고, 우측 1/3에 리더보드 카트리지 패널(`--surface-2` 그린) — 헤더 `RANKING · N분반`(픽셀폰트) + 1.5 테이블 스타일 TOP3(`lb-top3`) + 하단 구분선 아래 내 등수 행(`lb-myrank`, `▶YOU` 오렌지 하이라이트). 데이터 없으면 테이블 자리에 `NO RECORDS YET` + 잠자는 8x8 고스트 스프라이트. 우상단 헤더: `{닉네임}님 안녕하세요`(Do Hyeon) + `로그아웃` 텍스트 버튼 + 설정 키캡.

### S3. 로그인 요구 모달 (`modal-login-required`)
- 기능: "온라인 게임은 로그인이 필요합니다!" 문구 / 모달 내 구글 로그인 → 성공 시 S6 직행 / 취소하기 → 닫기. ESC/배경 클릭=취소.
- 레이아웃: 소형 모달 카드(360px). 타이틀 바 `LOGIN REQUIRED!`(픽셀폰트, 레드) + 자물쇠 8x8 스프라이트, 본문 Do Hyeon 문구, 아래 구글 로그인 버튼(풀폭) / `취소하기` 보조 버튼. 등장 시 스냅 팝 + 1프레임 셰이크(거부감 연출).

### S4. 설정 모달 (`modal-settings`)
- 기능: 라운드 수 / 라운드 당 시간(기본 3라운드·60초) 숫자 입력 + 단위(round/초) / 확인(`btn-settings-save`) / 기본값. 배경 클릭=저장 안 함. 저장값은 게임 라운드에 실제 반영.
- 레이아웃: 옵션 화면 오마주 — 타이틀 바 `OPTIONS`(픽셀폰트). 각 항목은 `▶ 라운드 수` 라벨 + 우측 스테퍼형 입력([-] 키캡 / 숫자 픽셀폰트 / [+] 키캡, 직접 타이핑 가능한 number input 겸용, min 1) + 단위 라벨. 하단 `확인`(오렌지) / `기본값`(그레이). 값 변경 시 숫자가 1프레임 옐로 플래시.

## §3. 게임 장면 아트 디렉션

### 3.1 게임1 — 숫자 펌핑 (오렌지/옐로 테마)
- 무대: 정사각 콘솔 스크린, 배경 `#000` + 하단 1/4에 오렌지 도트 지평선. 중앙 상단에 타겟숫자가 **거대 픽셀폰트 옐로(72px) + 검정 판넬**로 군림 — 판 전체가 "이 숫자를 맞춰라" 전광판.
- 펌핑 시각화: 좌(블루)/우(레드) 하단에 각자의 현재숫자 픽셀폰트 40px + 그 뒤로 **세로 픽셀 게이지 탑**(8px 블록 스택)이 현재숫자/100 비율만큼 진영색 블록으로 쌓인다. 타겟 높이에 옐로 점선 마커 — 숫자와 게이지가 이중으로 근접도를 보여준다.
- 캐릭터: 게이지 옆 8x8 픽셀 펌프맨(진영색)이 키 입력마다 펌프질 2프레임 애니. 연타 속도 빠르면 땀방울 스프라이트.
- 판정 연출: 타겟 일치 순간 게이지 상단에 옐로 별 반짝 + 유지 타이머가 **3칸 하트 도트**로 1초마다 채워짐(주변에서 유지 상태가 읽히는 긴장 장치). 이탈 시 하트가 픽셀 파편으로 깨짐. 3초 완성 → 화면 플래시 + 승자 게이지에서 별 폭죽 3프레임.
- 키패드: 하단 중앙에 Q(↓)/W(↑)·U(↓)/I(↑) 키캡 4개, 소유 진영색 링.

### 3.2 게임2 — 총알 (블루/라벤더 테마)
- 무대: 정사각 스크린, 배경 `#000` + 라벤더(#83769C) 별 2px 도트가 느리게 스크롤(우주 슈팅 게임 인용). 상단에 블루 트랙 라인(P1 존), 하단에 레드 존 라인(P2 존) — 트랙은 2px 라벤더 점선.
- 캐릭터: P1(블루)은 하향 포신 달린 8x8 픽셀 UFO — 자동 왕복은 2프레임 부스터 애니, 방향 반전 시 스프라이트 좌우 플립 + `!` 말풍선 1프레임. P2(레드)는 8x8 러너 — 이동 시 2프레임 발구름.
- 총알: 4x6 픽셀 캡슐(화이트 몸통 + 옐로 머리), 낙하 중 2px 잔상 점 1개. 속도 랜덤은 잔상 길이로 암시(빠른 탄=잔상 2개).
- 장전/쿨다운: P1 포신 아래 반투명 흐린 캡슐(다음 탄) — 쿨다운 중 그레이, 장전 완료 시 옐로 플래시. 와이어프레임의 "흐린 캡슐" 해석 계승.
- 피격: 3프레임 픽셀 폭발(레드→오렌지→옐로 파편) + 1프레임 화면 셰이크 + P2 스프라이트가 고스트로 팔레트 스왑. 생존 승리는 P2 주위 그린 별 궤도.
- 키패드: 좌하단 Q(⇄반전)/W(●발사) 블루 링, 우하단 U(←)/I(→) 레드 링.

### 3.3 게임3 — 펜싱 (그린/레드 테마 · 칸+바다)
- 무대: 와이드 스크린. 중앙에 **그린(#00E436) 픽셀 플랫폼**이 칸 단위로 분절(칸 경계는 검정 2px 홈, 칸마다 눈금 도트) — 좌우 각 3칸+중앙 대치 칸 구조가 한눈에 읽힘. 플랫폼 좌우 끝 바깥은 **블루(#29ADFF) 픽셀 바다** — 2프레임 파도 타일 애니 + 화이트 물거품 도트. 하늘은 `#000` + 라벤더 별.
- 캐릭터: 16x16 픽셀 펜서 2명(좌=블루, 우=레드), 3포즈 스프라이트 — 대기(검 세움)/공격(찌르기 런지 2프레임)/회피(방패 웅크림). 무행동 틱은 대기 포즈 + `zzz` 도트.
- 틱 공개 연출: 매 1초 틱마다 양측 머리 위에 선택 아이콘 말풍선(검/방패/`…`)이 동시 팝 → 상성 승자 쪽으로 `>` 화살표 플래시 → 패자가 1칸 뒤로 **step(2) 넉백**(칸 단위 스냅, 착지 먼지 도트). 같은 행동이면 `CLANG!` 옐로 스파크가 중앙에 팝.
- 위기 표현: 남은 칸 1칸이 되면 그 뒤 바다가 레드 틴트 파도로 팔레트 스왑 + 캐릭터 발밑 칸 점멸 — 링아웃 임박 경고.
- 링아웃: 캐릭터가 바다로 2프레임 낙하 + 화이트 물기둥 스플래시 3프레임 + `SPLASH!` 픽셀 텍스트. 승자는 검 들어올리는 승리 포즈.
- 카운트다운: 스크린 상단 중앙 픽셀 블록 바. 키패드: 좌하단 Q(⚔)/W(🛡)-블루, 우하단 U(⚔)/I(🛡)-레드 — 아이콘은 8x8 자체 도트 스프라이트(이모지 미사용).
