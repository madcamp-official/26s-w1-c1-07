## §1. 디자인 시스템

### 1.1 컨셉 축 & 차별화
- **밝음 / 맥시멀(고밀도) / 현대(학교 정서) / 장난스러움.** 지금까지의 루프 1~6은 전부 미니멀~중밀도였다. 이 시안이 미커버였던 **맥시멀리즘(고밀도)** 축을 담당한다.
- 밀도의 정체성: 화면 어디를 봐도 스티커 · 테이프 · 손글씨 낙서 · 도장 자국이 레이어로 겹쳐 "학급 공용 노트"처럼 지저분하게 꽉 찬다. 단, **정보 위계는 흐트러지지 않는다** — 낙서는 배경/장식 레이어, 실제 조작 요소(버튼·입력·숫자)는 흰 다이컷 스티커로 한 단 위에 떠 있어 또렷하다.

> 접근성: 파랑/빨강은 명도 대비가 충분(둘 다 크라프트 위 4.5:1↑)하고, 색맹 대비를 위해 P1은 **각진 사각 배지**, P2는 **동그란 배지** 등 형태로도 이중 인코딩한다.

### 1.3 타이포그래피 (구글 폰트 2종)

1. **Gaegu** (한글 지원 · 손글씨 디스플레이) — 헤드라인, 버튼 라벨, 손글씨 주석, 승패 문구, 한글 본문 전반. 어린이 공책 손글씨 느낌으로 컨셉의 중심.
2. **Fredoka** (라운드 산세리프 · 라틴/숫자) — "MADPUMP" 로고, HUD 숫자(타겟숫자·현재숫자·카운트다운), 리더보드 수치, 방 코드. 굵고 둥근 지오메트릭이라 손글씨와 대비되며 숫자 가독성을 지킨다.

`index.html` `<head>`에 넣을 link 태그:
```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Gaegu:wght@300;400;700&family=Fredoka:wght@400;500;600;700&display=swap" rel="stylesheet">
```

타입 스케일 (rem, 1rem=16px):
- 로고 MADPUMP: Fredoka 700, clamp(2.6rem, 7vw, 5rem), letter-spacing -0.01em, 볼펜 블루, 살짝 -2° 기울임.
- 화면 타이틀(설정/온라인 게임하기 등): Gaegu 700, 2rem.
- 버튼 라벨: Gaegu 700, 1.35rem.
- 본문/주석: Gaegu 400, 1.05rem.
- HUD 대형 숫자(타겟숫자): Fredoka 700, clamp(3rem, 9vw, 6.5rem).
- 리더보드 수치: Fredoka 600, 1.1rem, 타뷸러 느낌(고정폭 정렬).
- 손글씨 주석(nudge): Gaegu 400, 1rem, -4°~4° 회전.

### 1.4 형태 언어 (radius / border / shadow)
- **스티커 다이컷**: 모든 인터랙티브 요소는 흰 종이 스티커. `border: 3px solid #FFFDF7`(흰 테두리) **바깥에** `outline: 2.5px solid #26251F`(잉크 윤곽) 이중선 → 다이컷 스티커 특유의 흰 여백 테두리. radius 14~20px(유기적으로 불규칙 허용).
- **오프셋 하드 섀도**: `box-shadow: 4px 5px 0 rgba(38,37,31,.85)` — blur 0, 손그림 그림자. 큰 카드는 6px 7px.
- **회전**: 정적 요소는 -3°~3° 사이 랜덤 미세 회전으로 "손으로 붙인" 느낌. 단 **입력 중인 폼/모달 본문은 0°에 가깝게**(가독성).
- **마스킹테이프 고정**: 카드/스티커 상단 모서리에 반투명 테이프 조각(`#F3E7C6` @92%, 살짝 회전, 양끝 톱니 clip-path)을 얹어 "붙여둔" 서사.
- **손그림 테두리**: 강조 프레임(리더보드 패널, 게임 스테이지)은 SVG `feTurbulence`+`feDisplacementMap`으로 살짝 흔들리는 잉크 라인, 또는 `border-image`에 손그림 박스 사용. 과하면 성능 부담 → 큰 프레임 1~2곳만.
- **키캡 토큰(Q/W/U/I)**: 실물 키보드 키캡 모양. 흰 상판 + 하단 3px 어두운 립(입체), 눌리면 `translateY(2px)` + 립 소멸. P1 키(Q/W)는 파랑 밑동, P2 키(U/I)는 빨강 밑동.
- **정(正)자 태리마크**: 점수 표기는 종이 귀퉁이에 손으로 그은 바를 정(正)자 획(형광 그린).

### 1.5 모션 규칙
- **스티커 철썩(slap-in)**: 등장 시 `scale(0.8) rotate(-6°)` → `scale(1.08)` → `scale(1)`, 240ms, cubic-bezier(.34,1.56,.64,1)(오버슈트). squash&stretch로 "붙는" 느낌.
- **도장 쾅(stamp)**: 승리/도장 요소는 `scale(1.5) rotate(8°)` → `scale(1)` 급감속 + 착지 순간 화면 미세 흔들림 2px, 먼지 파티클.
- **hover wiggle**: 버튼/카드 hover 시 `rotate(-2°→2°)` 200ms 왕복 1~2회 + 섀도 6px로 확대.
- **키캡 press**: 실제 q/w/u/i 입력·클릭에 2px 가라앉음(50ms).
- **토스트**: "코드 복사됨!" 형광 그린 스티커가 아래에서 철썩 올라와 1.6초 후 팝-아웃.
- **전환**: 화면 전환은 노트 페이지 넘김 느낌(살짝 밀려 나가며 다음 페이지가 철썩). 모달은 스티커 slap-in + 배경 크라프트 딤(어둡게 대신 살짝 흐릿+채도↓).
- **접근성**: `prefers-reduced-motion` 시 회전/오버슈트/흔들림 제거, 페이드만.

### 1.6 컴포넌트 스타일 지침
- **버튼**: 흰 다이컷 스티커 + 이중 윤곽 + 오프셋 하드 섀도. 1차(주요 진입: 온라인/오프라인/빠른시작/확인) = 볼펜 블루 채움 + 흰 손글씨 라벨. 2차 = 흰 바탕 + 잉크 라벨. 위험/취소 = 흰 바탕 + 연필 회색 라벨. hover wiggle, press 시 섀도 소멸 + `translate(4px,5px)`(눌림). 각 버튼 살짝 다른 회전각.
- **카드**: 흰 종이 스티커 카드 + 마스킹테이프 상단 고정 + 오프셋 섀도. 게임 선택 카드는 폴라로이드/카드 진열처럼 겹쳐 흩뿌림.
- **모달**: 배경은 크라프트 딤(채도↓ + 8% 잉크). 모달 본문은 큰 흰 종이 스티커(테이프 4모서리 고정), slap-in 등장. 닫기는 우상단 손그림 X 스티커. ESC/배경 클릭 = 취소.
- **입력창**: 공책 밑줄 스타일 — 흰 스티커 안에 형광 옐로 밑줄 1개(밑줄만, 박스 최소화). 포커스 시 밑줄이 볼펜 블루로 바뀌고 굵어짐 + 커서 옆 깜빡이는 연필 아이콘. placeholder는 연필 회색 손글씨.
- **테이블(리더보드)**: 공책 위 손그림 표. 행 구분선은 형광펜으로 죽 그은 줄, TOP3는 금/은/동 대신 **1·2·3등 손그림 메달 스티커**. 수치는 Fredoka로 또렷하게. 내 등수 행은 형광 옐로 하이라이트 + "나!" 화살표 스티커.

## §2. 페이지별 계획 (S1~S12)

각 항목: SPEC 기능 요약(전부 존재해야 함) + 이 컨셉의 레이아웃/표현. data-testid는 레지스트리 준수. 디버그 브리지 `window.__MADPUMP__`를 화면 전환·게임 틱마다 갱신.

### S1 — 메인 · 비로그인 `scr-main-out`
- **기능(SPEC)**: 대형 타이틀, SIGN IN WITH GOOGLE(`btn-google-login`), 설정(`btn-settings`), 온라인 게임하기(`btn-online`, 비로그인 시 S3 가드), 오프라인 게임하기(`btn-offline`→S8).
- **표현**: 크라프트 게시판. 화면 중앙에 살짝 기운 거대한 **MADPUMP** 볼펜 로고 스티커. 로고 주변으로 아령·호루라기·별·"오늘의 대결!" 손글씨 낙서가 콜라주로 흩뿌려짐. 우상단 = 구글 로그인(구글 G 로고 포함) 스티커 + 톱니 설정 스티커. 중앙 하단 세로 스택으로 온라인/오프라인 두 개의 큰 스티커 버튼(온라인=파랑 채움, 오프라인=흰 바탕). 오프라인 버튼 옆엔 "로그인 안 해도 돼!" 손글씨 화살표. 온보딩 튜토리얼 없이 60초 진입.

### S2 — 메인 · 로그인 후 `scr-main-in`
- **기능(SPEC)**: 닉네임 인사말, 로그아웃, 우측 분반 리더보드(TOP3 플레이수·승리수·승률 + 내 등수, 빈 상태 정직 표시 `lb-top3`/`lb-myrank`), 온라인/오프라인 진입, 설정.
- **표현**: 좌측 = MADPUMP 로고 + "○○○님 안녕하세요" 손글씨 리본 스티커 + 로그아웃 텍스트 스티커 + 설정 톱니. 우측 = **공책 리더보드 패널**: 마스킹테이프로 붙인 "<○분반 내 리더보드>" 제목표, TOP3는 손그림 메달 스티커(1·2·3등) 행에 닉네임/플레이수/승리수/승률, 그 아래 점선 구분 후 "내 등수" 행을 형광 옐로 하이라이트 + "나!" 화살표. **mock 데이터 없으면** 가짜 순위 대신 "아직 기록이 없어요 ✏️" 빈 상태 손글씨(fallback 정직성). 하단 중앙 온라인/오프라인 버튼.

### S3 — 로그인 요구 모달 `modal-login-required`
- **기능(SPEC)**: "온라인 게임은 로그인이 필요합니다!" 문구, 모달 내 구글 로그인(`btn-google-login`)→성공 시 S6, 취소하기→S1.
- **표현**: S1 위에 크라프트 딤. 중앙에 흰 종이 스티커 모달(테이프 4모서리), 상단에 "잠깐! 🔒" 손그림. 문구 아래 구글 로그인 스티커 버튼 + "취소하기" 흰 버튼. 배경/ESC = 취소.

### S4 — 설정 모달 `modal-settings`
- **기능(SPEC)**: 제목 "설정", 라운드 수 입력(단위 round), 라운드 당 시간 입력(단위 초), 확인(`btn-settings-save`)/기본값. 기본값 = 라운드 3 / 시간 60초(SPEC Q1). S6 톱니에서도 재사용.
- **표현**: 흰 종이 스티커 모달. 두 입력은 공책 밑줄 필드 + 옆에 "round" "초" 손글씨 단위. 값은 Fredoka 큰 숫자. 하단 "확인"(파랑) / "기본값"(흰) 스티커 버튼. 기본값 클릭 시 값만 리셋(모달 유지). 배경 클릭 = 저장 안 함. 저장 라운드 수는 이후 게임에 실제 반영(QA-S4-06).

### S5 — 닉네임 온보딩 `scr-onboarding`
- **기능(SPEC)**: "What's your name?" 카드, 이름(`input-nickname`)·분반 입력, 확인(`btn-nickname-submit`), 이름 중복 검증→에러 `err-nickname-dup`("이미 사용하고 있는 이름입니다"), 통과 시 S2. 금지 이름 배열에 "test" 포함. 빈 입력 제출 방지.

## §3. 게임 장면 아트 디렉션

공통 규칙: shared 순수 로직의 state를 **그대로 렌더링만** 한다(로직 재구현 금지). 모든 게임은 "공책 한 페이지" 프레임 안에서 벌어지고, 나=형광 옐로 하이라이트 + "나!" 화살표로 항상 식별된다.

### 게임1 — 숫자 펌핑 (온도계 낙서)
- **핵심 은유**: 펌핑 = 손그림 **온도계에 형광펜이 차오름**. 숫자 UI(Fredoka)와 낙서 일러스트를 통합해 "얼마나 왔나"가 즉각 보인다.
- 중앙 타겟숫자는 형광 옐로 원에 볼펜으로 동그라미 친 "목표!". 각 온도계 눈금에 타겟 라인이 빨강 채점펜으로 그어져 있어 근접도 비교.
- 일치 순간: 온도계 구슬이 형광색으로 부풀며 3초 게이지 링 충전. 3초 달성 = "POW!" 만화 폭발 스티커 + 도장 쾅. 이탈 시 링이 지우개로 슥 지워지며 리셋.
- 오버슈트(타겟 초과): 온도계 밖으로 형광펜이 삐져나가며 "앗 넘었다!" 손글씨.
- 승수는 종이 귀퉁이 정(正)자 획으로 누적.

### 게임2 — 총알 (잉크 방울 낙하)
- **핵심 은유**: 총알 = 볼펜 촉에서 **떨어지는 잉크 방울/압정 낙서**. 낙하하며 잉크 번짐 트레일.
- P1(파랑, 상단 각진 배지)은 연필 왕복 자국을 남기며 자동 이동, 발사 시 촉에서 잉크 방울 툭. 쿨다운은 키캡 위 형광 오렌지 "장전" 게이지.
- P2(빨강, 하단 동그란 배지)는 좌우로 미끄러지며 회피. 아슬한 회피엔 "휘익~" 효과선 + 형광 옐로 하이라이트 호.
- 피격: 잉크 방울이 P2에 철썩 → 잉크 대폭발 튐 + "퍽!" 스티커 → P1 라운드 승. 시간 생존 = P2에 "SAFE!" 형광 그린 도장.
- 속도 랜덤(shared state 반영)이라 방울마다 낙하 속도/번짐 크기가 다름.

### 게임3 — 펜싱 (공책 체육관 + 파도 바다)
- **핵심 은유**: 좌우 손그림 검객이 1초 틱마다 가위바위보(⚔️/🛡️/💤)로 상대를 **파도 바다 쪽으로 밀어내기**.
- 무대는 볼펜 플랫폼, 좌우 끝 형광 시안 파도 낙서. 바닥 칸 눈금으로 남은 거리(시작 3칸)를 표시 — shared state의 distanceFromEdge를 칸으로 그대로 매핑.
- 매 틱 선택은 검객 머리 위 말풍선 스티커로 철썩 공개(동시). 검객 포즈는 스탠스에 따라 검 찌르기/방패 들기/멀뚱(무행동)으로 교체.
- 판정: 진 쪽 검객이 만화 효과선 휘날리며 1칸 미끄러짐(먼지 스티커). 같은 행동이면 "챙!" 검 부딪힘 이펙트만(밀림 없음).
- 링아웃: 벼랑 끝 넘으면 "풍덩! 🌊" 물기둥 + 검객이 파도에 빠지는 squash 애니메이션 → 라운드 패배.
- 시간 종료: 더 밀린(칸 적은) 쪽 패배, 동률 무승부 — 상단 카운트다운 하이라이터 바 소진으로 표시.
