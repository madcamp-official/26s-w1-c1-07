# 16:1293 — 온라인 게임하기 - 로그인이 안되어있을때 (실제 내용: 설정 모달)

- 이미지: `design-lab/reference/figma/16-1293__온라인_게임하기_-_로그인이_안되어있을때.png`
- 캔버스 위치: (-5163, 2563)

## 역할 추정
프레임 이름은 16:1276과 같지만 **실제 내용은 "설정" 모달**이다. 메인 화면 위에 게임 규칙 설정(라운드 수, 라운드 당 시간) 모달이 덮여 있다. 우상단 "설정" 원형 버튼 또는 톱니 아이콘을 눌렀을 때의 상태로 추정. 비로그인 상태에서도 접근 가능(우상단에 SIGN IN 버튼이 그대로 보임).

## 구성요소 (위치 포함)
- **배경**: 메인 화면 잔재 — 우상단 "SIGN IN WITH GOOGLE", 원형 "설정" 버튼, 하단에 가려진 "오프라인 게임하기" 버튼 일부, 톱니 아이콘
- **중앙 대형 모달 (회색 박스)**:
  - 제목: "설정" (모달 상단 중앙)
  - 라벨: "라운드 수" / 그 아래 숫자 입력창(값 "3") + 단위 텍스트 "round"
  - 라벨: "라운드 당 시간" / 그 아래 숫자 입력창(값 "3") + 단위 텍스트 "초"
  - 하단 버튼 2개(가로 나란히): "확인", "기본값"

## 기능 목록
1. **라운드 수 설정**
   - (a) 대결 라운드 수 지정 (기본 3).
   - (b) 트리거: 숫자 입력창 편집.
   - (c) mock: number input, min/max 검증, 상태 저장.
2. **라운드 당 시간 설정**
   - (a) 각 라운드 제한 시간(초) 지정 (기본 3초 — 매우 짧은 반응형 미니게임임을 시사).
   - (b) 트리거: 숫자 입력창 편집.
   - (c) mock: number input.
3. **확인(저장)**
   - (a) 설정값 저장 후 모달 닫기. 이후 게임 시작 시 이 값 사용.
   - (b) 트리거: "확인" 클릭.
   - (c) mock: localStorage 또는 전역 store에 `{rounds, secondsPerRound}` 저장.
4. **기본값 복원**
   - (a) 두 값을 기본값(3 round / 3초)으로 리셋.
   - (b) 트리거: "기본값" 클릭.
   - (c) mock: 상태 리셋 (모달은 열린 채 유지가 자연스러움).
5. **설정 모달 열기/닫기** (전제)
   - (b) 트리거: 메인의 "설정" 버튼/톱니 아이콘 클릭. 닫기는 "확인" (별도 X 버튼 없음).

## 특이사항
- **프레임 이름 오기(誤記)로 보임**: 이름은 "온라인 게임하기 - 로그인이 안되어있을때"인데 내용은 설정 모달. 로그인 요구 모달은 16:1276 쪽이 진짜.
- 취소 버튼이 없고 "확인/기본값"만 있음 — 취소 없이 항상 저장하는 단순 구조. 구현 시 배경 클릭 닫기 = 저장 안 함으로 할지 결정 필요.
- 이 설정이 온라인/오프라인 공통 설정인지, 방장만 설정 가능한지는 와이어프레임만으로 불명 — 기능정의서 대조 필요.
