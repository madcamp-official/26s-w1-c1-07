# 16:2062 — 그룹 관리 - 랭킹 조회

- 이미지: `design-lab/reference/figma/16-2062__그룹_관리_-_랭킹_조회.png`
- 캔버스 위치: (4827, 7405)

## 역할 추정
관리자용 "그룹 관리 → 특정 그룹(몰입캠프 1분반) → 랭킹 조회" 탭. 그룹 내 멤버들의 점수(랭킹) 정보를 표 형태로 조회하는 화면. 인원 탭과 거의 같은 레이아웃에 "점수" 열이 추가된 형태.

## 구성요소
- 좌측 사이드바: "MADPUMP" / "그룹 관리" / "전체 인원 관리" / 하단 "로그아웃"
- 본문 상단: "◀ 뒤로가기" / 타이틀 "몰입캠프 1분반"
- 탭 3개: "인원" / "매치 결과 관리" / "랭킹 조회"(활성, 검정 배경)
- 검색 줄: 라벨 "닉네임 검색" + 텍스트 입력창 + 돋보기 아이콘
- 테이블 (열): "닉네임" / "Google ID" / "가입일시" / "플레이 수" / "점수" / "관리"
  - 행 3개: "박준서 / asdferq@google.com / 2026-07-02 19:00 / (플레이수 빈칸) / (점수 빈칸) / 내보내기(빨간 링크)"
  - "이종혁", "라태형" 동일 패턴 + 빈 행 다수

## 기능 목록
1. 그룹 내 랭킹(점수) 목록 조회
   - (a) 그룹 멤버별 닉네임/Google ID/가입일시/플레이 수/점수 표시
   - (b) 랭킹 조회 탭 진입 시 자동 로드
   - (c) mock: 멤버 mock 배열을 점수 내림차순 정렬해 렌더 (와이어프레임에는 점수 값·정렬 표시가 비어 있으나 랭킹이므로 점수 정렬이 자연스러움)
2. 닉네임 검색
   - (a) 닉네임으로 목록 필터
   - (b) 입력 후 돋보기 클릭(또는 엔터)
   - (c) mock: 문자열 includes 필터
3. 멤버 내보내기
   - (a) 그룹에서 멤버 제외
   - (b) 관리 열 빨간 "내보내기" 클릭
   - (c) mock: 확인 모달(16:1780 "내보내기 버튼 눌렀을 때" 프레임 참고) 후 배열에서 제거
4. 탭 전환 / 뒤로가기 / 사이드바 내비 / 로그아웃

## 특이사항
- "랭킹 조회" 탭인데 순위(등수) 열이 따로 없고, 인원 탭 테이블에 "점수" 열만 추가한 구성. 관리 열의 "내보내기"까지 그대로 남아 있음 — 인원 탭 프레임을 복제·변형한 와이어프레임으로 보임. 구현 시 랭킹 탭에서 내보내기를 유지할지 재검토 필요.
- 16:2193(그룹 내 인원, 검색 결과)과 테이블 열 구성이 동일(점수 열 포함) — 두 프레임이 사실상 같은 표를 공유.
- Google ID가 모든 행 동일("asdferq@google.com") — placeholder 데이터.
