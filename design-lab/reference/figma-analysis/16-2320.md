# 16:2320 — "관리자 메인페이지" (실제 내용: 관리자 로그인 화면)

## 역할 추정
프레임 이름은 "관리자 메인페이지"지만 실제 내용은 **관리자 로그인 화면**. ID/PW 입력 후 관리자 대시보드(16:1716)로 진입하는 관리자 플로우의 진입점.

## 구성요소 (위치와 함께)
- **상단 중앙**: 로고 텍스트 `MADPUMP` (굵은 글씨)
- **중앙**: 로그인 폼 (세로 배치, 중앙 정렬)
  - 라벨 `ID` (좌측) + 회색 입력창(우측, 가로로 긴 직사각형)
  - 라벨 `PW` (좌측) + 회색 입력창(우측, 동일 크기)
- 로그인/제출 버튼은 와이어프레임상 **보이지 않음** (생략된 것으로 추정 — Enter 제출 또는 이후 추가 예정)

## 기능 목록
1. **ID 입력**
   - (a) 관리자 계정 아이디 입력
   - (b) 트리거: 입력창 포커스 후 타이핑
   - (c) mock: `<input type="text">`, 상태 바인딩
2. **PW 입력**
   - (a) 관리자 비밀번호 입력(마스킹)
   - (b) 트리거: 입력창 포커스 후 타이핑
   - (c) mock: `<input type="password">`
3. **로그인 제출**
   - (a) 자격 증명 검증 후 관리자 대시보드(16:1716)로 이동
   - (b) 트리거: Enter 키 또는 (와이어프레임에 없는) 로그인 버튼
   - (c) mock: 하드코딩 계정(admin/admin 등) 비교 → 성공 시 `/admin` 라우트, 실패 시 인라인 에러 메시지. 버튼이 없으므로 mock에는 명시적 로그인 버튼을 추가하는 것을 권장.

## 특이사항
- **이름-내용 불일치**: 이름은 "관리자 메인페이지"인데 내용은 로그인 폼. 같은 이름의 16:1716이 진짜 메인(사이드바 대시보드)이다. 두 프레임은 관리자 플로우의 연속 단계(로그인 → 메인)로 추정.
- 로그인 버튼, 에러 메시지, 아이디 저장 등의 부가 요소는 전혀 없음 — 최소 골격 와이어프레임.
