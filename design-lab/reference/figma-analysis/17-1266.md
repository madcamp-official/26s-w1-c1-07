# 17:1266 — "게임2" (게임2 펜싱 대결 — 16:1573의 복제 프레임)

이미지: `design-lab/reference/figma/17-1266__게임2.png`

## 역할 추정
**16:1573과 시각적으로 동일한 복제 프레임.** 캔버스에서 다른 위치(-2121,9943)에 놓인 사본으로, 다른 플로우 열(row)에서 같은 화면을 참조하기 위해 복제한 것으로 추정. 게임2 펜싱 대결 인게임 화면.

## 구성요소 (16:1573과 동일)
- 상단 좌: 원형 버튼 **"변경"**
- 상단 우: 원반 패드 **"P"** + 겹쳐 내려오는 아래 화살표(연타 표현)
- 중앙: 좌측 검 든 캐릭터(공격), 우측 방패 든 캐릭터(방어)
- 하단: 무대 플랫폼, 내부에 텍스트 **"(count down)"**

## 기능 목록
16:1573 분석(`16-1573.md`)과 동일:
1. P 키 연타 (공격 게이지) — keydown `P` / mock: 연타 카운트 게이지
2. "변경" 스탠스 전환 — Q 키 추정 / mock: 토글 + 스프라이트 스왑
3. 캐릭터 스탠스 시각화 — 상태 변경 시 / mock: 이미지 교체
4. 카운트다운 — 라운드 시작 시 / mock: setInterval
5. 공격 vs 방어 판정 — 게이지/상성 / mock: 봇 시뮬레이션
