# 17:1302 — "게임2" (게임2 펜싱 대결 — 16:1607의 복제 프레임)

이미지: `design-lab/reference/figma/17-1302__게임2.png`

## 역할 추정
**16:1607과 시각적으로 동일한 복제 프레임.** 캔버스 다른 위치(-2121,11106)의 사본. 게임2 펜싱 대결에서 양 플레이어의 Q 패드 + 현재 장비(검/방패)를 보여주는 상태 도식.

## 구성요소 (16:1607과 동일)
- 상단 좌: 원반 패드 **"Q"** + 검 아이콘 (좌측 플레이어 = 공격 스탠스)
- 상단 우: 원반 패드 **"Q"** + 부채꼴 방패 아이콘 (우측 플레이어 = 방어 스탠스)
- 중앙: 좌측 검 든 캐릭터, 우측 방패 든 캐릭터
- 하단: 빈 무대 플랫폼 (count down 텍스트 없음)

## 기능 목록
16:1607 분석(`16-1607.md`)과 동일:
1. Q 키 스탠스 전환 — keydown `Q` / mock: attack↔defense 토글
2. 플레이어별 스탠스 인디케이터 — 상태 변경 시 아이콘 스왑
3. 캐릭터 포즈 동기화 — 상태별 SVG 교체
4. 공격 vs 방어 상성 판정 — 스탠스 조합 테이블
