# 23:2529 — 메인 - 로그인 되기 이전 (변형/초안)

- 이미지: `design-lab/reference/figma/23-2529__메인_-_로그인_되기_이전.png`
- 캔버스 위치: (-5475, 8809) — 다른 프레임들보다 훨씬 아래쪽(별도 구역)에 위치

## 역할 추정
이름은 16:1264와 같지만 내용이 전혀 다르다. 로그인/설정/온·오프라인 분기가 전부 없고 **게임 종류 선택 버튼 3개만 있는 화면**. 두 가지 해석 가능:
1. 초기 버전(v1) 메인 화면 초안 — 게임만 바로 고르는 단순 구조였다가, 이후 온라인/오프라인 분기 + 로그인 구조(16:1264)로 발전.
2. 오프라인 게임하기를 눌렀을 때 나오는 "게임 선택" 화면 스케치.
캔버스에서 다른 프레임들과 동떨어진 위치인 점을 보면 1번(구버전 잔재) 가능성이 높음.

## 구성요소 (위치 포함)
- **중앙 상단**: 대형 타이틀 "MADPUMP"
- **중앙 하단 (가로 나란히 3개 버튼)**:
  - "게임1" (회색 사각 버튼)
  - "게임2" (회색 사각 버튼)
  - "게임3" (회색 사각 버튼)
- 그 외 요소 없음 (로그인 버튼, 설정 버튼 없음)

## 기능 목록
1. **게임1 선택** / 2. **게임2 선택** / 3. **게임3 선택**
   - (a) 해당 게임(게임1=17:1336, 게임2=17:1234, 게임3=17:1424 프레임 참고)으로 진입.
   - (b) 트리거: 각 버튼 클릭.
   - (c) mock: 게임별 라우트로 이동. 오프라인 모드라면 같은 기기 2인 대결 화면으로.

## 특이사항
- **16:1264와 동명이지만 내용 상이** — 이 프레임을 최종 메인으로 쓰면 안 되고, "게임 3종 선택" 개념만 참고하면 됨.
- 최종 플로우에서는 게임 선택이 어디서 일어나는지(방 생성 시? 매칭 후? 오프라인 진입 후?)를 기능정의서와 대조해 확정할 필요가 있음. 이 프레임은 그 선택 UI의 유일한 스케치라는 점에서 가치가 있음.
