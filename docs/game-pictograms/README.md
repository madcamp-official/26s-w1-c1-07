# 게임 4~10 대표 픽토그램 컨셉 탐색 (70종)

오프라인 게임 선택 화면(`client/src/screens/GameSelect.tsx`)의 게임 4~10 캐비닛 스크린에
들어갈 **대표 픽토그램** 디자인 탐색 아카이브.

## 무엇인가
- 게임 4~10 각 게임마다 **서로 다른 10가지 시각 전략**(히어로 스프라이트 / 액션 순간 /
  탑다운 / 사이드뷰 / HUD·게이지 / 엠블럼 / 타이포 / 추상 기하 / 환경 / 스플릿 듀얼)으로
  컨셉 10개씩 → **총 70종**.
- 각 시안은 **작업 → QA → 개선 5회 루프**를 거쳤고, QA 루브릭에 **이전 디자인들과의 유사도
  (0~100)** 항목이 있어 유사도가 높으면(>55) 마크를 새로 기획하도록 강제했다.
  최종 유사도 평균 ≈ 31~33.
- 모든 픽토그램은 `theme.css`(NEON COIN-OP) 팔레트·글로우 규칙을 따르는 SVG 내부 마크업이며,
  `<svg viewBox="0 0 120 108">` 안에 그대로 삽입해 렌더한다. 색은 `gp-*` 클래스로 지정.

## 파일
- `concepts.json` — 70종 전체(게임별 브리프 + 각 디자인의 finalSvg/firstDraftSvg/qaLog(5)/selfScores).
- `gallery.html` + `data.js` — 실제 인게임처럼(아케이드 폰트 + CRT 스캔라인 + 베젤) 70종을 보는 리뷰 갤러리.
  로컬 정적 서버로 서빙: `python3 -m http.server` 후 `gallery.html` 열기(구글 폰트 로드 위해 온라인 필요).
- `workflow.mjs` — 70종을 생성한 다중 에이전트 워크플로 스크립트(재현/재실행용).
- `build_data.py` — 워크플로 journal → `data.js`(갤러리 데이터) 재구성 스크립트.
